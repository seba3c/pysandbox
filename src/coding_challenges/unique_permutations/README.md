# Unique Permutations

Given a number, return the number of distinct permutations with same digit length as input number
that represent an integer number with no leading zeros. Example:

```
unique_permutations(123) => 6
valid 3 length permutations are: 123, 132, 213, 231, 312, 321
```

```
unique_permutations(100) => 1
valid 3 length unique permutation with no leading zeros are: 100
```

```
unique_permutations(11) => 1
valid 2 length unique permutation are: 11
```

```
unique_permutations(0) => 1
valid 1 length unique permutation are: 0
```
